# ♟️ C++ Terminal Chess

A fully functional terminal-based chess game built in C++ using Object-Oriented Programming principles.

This project implements complete chess mechanics including all standard chess rules, move validation, check/checkmate detection, castling, en passant, pawn promotion, and stalemate detection.

---

# 🚀 Features

## ✅ Complete Chess Rules
- Legal piece movement
- Turn-based gameplay
- Capturing system
- Illegal move prevention

## ♔ Special Chess Moves
- Castling
- En passant
- Pawn promotion

## ⚔️ Game State Detection
- Check detection
- Checkmate detection
- Stalemate detection
- Draw by threefold repetition
- Draw by 50 move rule

## 🧠 OOP Concepts Used
- Inheritance
- Polymorphism
- Virtual functions
- Dynamic memory allocation
- Class-based architecture

# ▶️ How to Run

## Compile

```bash
g++ chess.cpp -o chess
```

## Run

```bash
./chess
```

# 🎮 Move Input Format

Moves are entered in standard coordinate format:

```text
e2e4
g1f3
e7e5
```

# ♟️ Example Gameplay

White's turn: e2e4

White's turn: g1f3
